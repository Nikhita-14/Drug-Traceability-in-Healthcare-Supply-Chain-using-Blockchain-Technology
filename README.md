# Drug Traceability in Healthcare Supply Chain using Blockchain Technology

## Overview
This project implements a blockchain-based system to enable secure and transparent tracking of pharmaceutical products across the healthcare supply chain. The system ensures data integrity, prevents counterfeit drugs, and allows stakeholders to verify the authenticity of medicines at every stage.

## Prerequisites
Ensure the following software is installed on your system:

- Python 3.7.0  
- Node.js v12.13.1 (64-bit)  

## Required Python Packages
Install the required dependencies using the following commands:

pip install web3==4.7.2  
pip install Django==2.1.7  
pip install numpy==1.19.2  

## Setup Instructions
1. Install Python 3.7.0 and configure environment variables.
2. Install Node.js v12.13.1 (64-bit).
3. Complete the Ethereum setup inside the `hello-eth` directory.
4. Install all required Python packages listed above.

## How to Run the Project
1. Open Command Prompt.
2. Navigate to the project directory:

cd Blockchain/DrugTracing

3. Start the application by executing:

run.bat

4. Access the application through the local server and use the interface to perform operations such as product registration and traceability.

## Project Structure
- `Blockchain/DrugTracing/` – Django backend and application logic  
- `Blockchain/hello-eth/` – Ethereum and blockchain-related setup  
- `templates/` – HTML templates  
- `static/` – CSS, JavaScript, and image assets  

## Notes
- Ensure all dependencies are installed before running the project.  
- The application relies on a properly configured blockchain environment in the `hello-eth` folder.  
- Default configuration uses a local development server.
